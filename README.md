### Show the distribution and correlation of nodes

requirements:
- [node](https://nodejs.org/en/) > 8.0.0 [download]
- [yarn](https://yarnpkg.com)


how to run:
1. install dependents `yarn`
2. dev `yarn start`
3. build `yarn build`
